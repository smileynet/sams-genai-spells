# Ritual: Challenge

**Spells:** blind-spot + zoom-out (parallel)
**Type:** Parallel-then-synthesize
**Plan mode:** Compatible

**Use when:** You need a comprehensive stress-test of a plan or direction. Blind-spot finds what you don't know you don't know; zoom-out challenges whether you're solving the right problem. Together they attack the plan from complementary angles — blind-spot probes for hidden gaps, zoom-out challenges the framing itself.

---

## Phase 1: Parallel Execution

Run both spells independently with the same topic:

**Spell A: blind-spot** — Run with the user's topic as the argument.
**Spell B: zoom-out** — Run with the user's topic as the argument.

**Full spell execution for both** — run each spell's entire process without modification.

**Execution note:** Run spell A first, then spell B. Both receive the original topic directly — no bridging between them.

---

## Synthesis

After both spells complete, produce a synthesis that cross-references their outputs:

**Extract** from blind-spot output:
- **BLIND SPOTS** — the numbered findings with categories and evidence
- **PATTERN MAP** — convergence clusters and technique yield

**Extract** from zoom-out output:
- **STRATEGIC FINDINGS** — the numbered findings with lenses and evidence
- **CONVERGENCE MAP** — cross-lens patterns
- **STRATEGIC RECOMMENDATION** — the verdict (stay course / adjust / pivot / stop)

**Cross-reference analysis:**

**Convergent findings** (same conclusion from different angles = strong signal):
Look for cases where blind-spot and zoom-out independently identified the same underlying issue. For example, if blind-spot found a "hidden assumption about user behavior" and zoom-out's reframing lens found "the problem definition assumes user behavior that may not hold" — that convergence from independent analyses is the strongest signal in the ritual.

**Divergent findings** (tensions requiring resolution):
Look for cases where the two analyses tension each other. For example, if zoom-out recommends "stay course" but blind-spot found critical unexamined failure modes — that tension needs explicit resolution: either the failure modes change the recommendation, or there are mitigations that preserve the direction.

**Emergent insights** (visible only in combination):
- Blind-spot's hidden assumptions may explain WHY zoom-out found certain strategic weaknesses
- Zoom-out's reframing may reveal that some of blind-spot's findings are symptoms of a deeper framing problem
- The combined coverage assessment: what did both analyses miss? (their shared blind spots)

**Combined verdict:**
Synthesize zoom-out's strategic recommendation with blind-spot's findings. If blind-spot found critical issues that zoom-out's recommendation doesn't account for, adjust the recommendation accordingly. Produce one clear recommendation, not two separate ones.

**Unified next steps:**
Merge and deduplicate the action items from both analyses. Group by priority, not by source spell.
